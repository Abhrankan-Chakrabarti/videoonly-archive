How to Submit

  1. Write your message locally (offline if possible).
  2. Encrypt the message using the public PGP key above.
  3. Send the encrypted message via email to videoonly.archive@protonmail.com.

At present, encrypted email is the only supported submission method.

Messages sent without encryption may be ignored.

Recommended Tools

  • OpenKeychain (Android)
  • Any PGP-compatible encryption tool

Important Notes
