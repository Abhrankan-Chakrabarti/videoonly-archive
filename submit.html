How to Encrypt a Message (Simple Guide)

You do not need technical knowledge to encrypt a message. Once encrypted, only the owner of this site can read it.

Option 1: Android (Easiest)

  1. Install OpenKeychain from F-Droid or Play Store
  2. Download the public PGP key from the link above
  3. Open OpenKeychain → Import Key → select the downloaded key
  4. Write your message in any notes app
  5. Share or copy the text → choose Encrypt
  6. Select the imported VideoOnly key
  7. Copy the encrypted text (it starts with -----BEGIN PGP MESSAGE-----)
  8. Email the encrypted text to the address above

Option 2: Computer (Linux / macOS / Windows)

  1. Install a PGP tool:
    • Linux: GnuPG (usually preinstalled)
    • macOS: GPG Suite
    • Windows: Gpg4win
  2. Download the public PGP key from this page
  3. Import the key: 
    gpg --import videoonly_publickey.asc
  4. Write your message in a text file (example: message.txt)
  5. Encrypt it: 
    gpg --encrypt --armor --recipient videoonly.archive@protonmail.com message.txt
  6. A file named message.txt.asc will be created
  7. Open it and send its contents via email

Important Notes

  • Only encrypted messages can be read
  • Unencrypted messages may be ignored
  • You may remain anonymous if you choose
  • Anyone intercepting the message will see only scrambled text
  • Not all submissions will be published
